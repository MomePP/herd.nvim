*herd.txt*                Drive herdr coding agents from Neovim

==============================================================================
INTRODUCTION                                                            *herd*

herd.nvim drives herdr (https://herdr.dev) coding agents from Neovim, for when
herdr is the host: nvim runs in one herdr pane and CLI agents (claude,
opencode, ...) run in sibling panes. Spawn a tool and zoom it fullscreen,
toggle to a live agent, and send the visual selection — mirroring the
sidekick.nvim UX without nvim hosting the agents.

It talks to the `herdr` CLI only; there is no embedded terminal.

==============================================================================
REQUIREMENTS                                               *herd-requirements*

- Neovim >= 0.10        (uses |getregion()|)
- herdr >= 0.7          on $PATH, with a running server you are attached to
- nvim launched inside a herdr pane (recommended)

==============================================================================
SETUP                                                             *herd.setup()*

>lua
    require('herd').setup({
      tools = {
        claude   = { cmd = { 'claude' } },
        opencode = { cmd = { 'opencode', '--continue' } },
      },
    })
<

Options ~                                                         *herd-config*

    tools   table<string, { cmd: string[], env?: table }>   (default: {})
            Spawnable agents. The key is the tool name; `cmd` is the argv.
    keys    { toggle, send, select }   (set any to false to disable)
            Default: toggle = false, send = "<leader><Tab>",
                     select = "<leader><Tab>"
    zoom    boolean   (default: true)
            Zoom the agent pane fullscreen on toggle/spawn.

==============================================================================
MAPPINGS                                                        *herd-mappings*

    <leader><Tab>   (normal)  select: switch to a running agent, or spawn a tool
    <leader><Tab>   (visual)  send the selection to the active agent
    toggle          (normal)  off by default; set keys.toggle to jump straight
                              to this cwd's agent / spawn if none

==============================================================================
NAVIGATION                                                       *herd-return*

herd is a spawner; moving nvim<->agent belongs to your multiplexer. The agent
is spawned as a split in nvim's tab (nvim-left / agent-right), so use herdr's
directional pane focus (tab-scoped, reliable across workspaces): >toml
    [keys]
    focus_pane_left  = "prefix+h"   # -> nvim
    focus_pane_right = "prefix+l"   # -> agent
<
Avoid herdr last_pane/cycle_pane_* here — they are global/cycle and won't
reliably return to this project's agent after switching workspaces.

==============================================================================
COMMANDS                                                        *herd-commands*

    :Herd [toggle|select|send]      default: toggle

==============================================================================
API                                                                  *herd-api*

    require('herd').toggle()
    require('herd').select()
    require('herd').send()
    require('herd').spawn(tool)

==============================================================================
HEALTH                                                            *herd-health*

    :checkhealth herd
